| PhpStorm 단축키
===================

1. 실행
```
★ Ctrl+Shift+F10 : 실행
Ctrl+F10 : 이전실행
Alt+Insert : 파일 관리 리스트 (마우스 우클릭 효과)
```
2. 라인 수정
```
★ Ctrl+d : 한 라인 복붙
★ Ctrl+y : 한 라인 삭제
Ctrl+Shift+j : 라인 합치기
Ctrl+Shift+↑(↓) : class안에서 한 라인 이동
Alt+Shift+↑(↓) : 전체안에서 한 라인 이동
```
3. 검색
```
Ctrl+f : 현재 파일안에서 검색
Ctrl+Shift+f : 전체 검색
Ctrl+Shift+n : 파일 검색
Ctrl+Shift+Alt+n : 메소드 검색
Ctrl+Shift+a : 액션 옵션 검색
Ctrl+e : 최근 열었던 파일 목록 검색
Ctrl+shift+e : 최근 수정했던 파일 목록 검색
★ Shift+Shift : IDE 전체 검색
Ctrl+r : 현재 파일안에서 치환
Ctrl+Shift+r : 전체 치환
```
4. 블록씌우기
```
Shift+→(←) : 블록씌우기
Ctrl+Shift+→(←) : 단어 단위로 블록씌우기
★ Ctrl+w : 단위 블럭씌우기
Ctrl+Shift+w : 단위 블럭해제
★ Ctrl+Ctrl+↑(↓) : 같은 라인 여러줄 블럭씌우기
```
5. 보기
```
Ctrl+p : 인자 타입 즉시보기
★ Ctrl+Shift+i : 코드 구현 즉시보기
★ Ctrl+q : DOC 즉시보기
Alt+→(←) : 소스파일 탭 이동
Alt+↑(↓) : 메소드 단위 포커스
★ F2 : 오류라인 자동 포커스
```
